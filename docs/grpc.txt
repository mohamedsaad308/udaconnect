#########################################################
#             How to make a sample request				#
#########################################################


1. Clone https://github.com/mohamedsaad308/udaconnect

2. Install and run python 3.8

3. Install the requirements found in `/udaconnect/modules/grpc/`

`pip install -r requirements.txt`

3. Go to `/udaconnect/modules/grpc` and Run writer.py

`python3 writer.py`
